HTML:-

    -> Hyper Text MarkUp Language is the expansion for HTML

    -> HTML is the standard markup language for Web pages.

    -> With HTML you can create your own Website.

Tags:-

    -> HTML Tag is commonly defined as a set of characters constituting a formatted command.

    -> Tags provide the directions or recipes for the visual content of the Web.

    ->The tag names are enclosed in between angle brackets or a ‘less than’ and a ‘greater than’ symbol, (<) and (>)

Types of Tags:- (common catagories)

  • Structure Tags
  • Paired & Unpaired Tags
  • Self-Closing Tags
  • Utility-Based Tags
  • Formatting Tags
  • Control Tags
  • Frame Tag and Iframes
  • Quotation Tag
  • Meta Tag

Structure Tags:-

    -> The Tags that help in structuring the HTML document are called Structure Tags. They are,


  1. <!doctype html>(Description Tag)

    • This tag represents the document type is HTML, and helps browsers to display web pages correctly.
    • It must only appear once, at the top of the page (before any HTML tags).
    • Tag declaration is case insensitive.

  2. <html> ... </html>

    • This is the root element of an HTML page
    • It the container for all other HTML elements(except for the <!DOCTYPE> tag).

  3. <head> ... </head>

    • This is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.
    • Metadata is data about the HTML document. Metadata is not displayed.
    • The elements inside the <head> element are:-
      1. <title> (required in every HTML document)
      2. <style>
      3. <base>
      4. <link>
      5. <meta>
      6. <script>
      7. <noscript>

  4. <body> ... </body>

    • The <body> tag defines the document's body.
    • This element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
    • This tag supports both Global and Event Attributes
    • There can only be one <body> element in an HTML document.

Unpaired or empty Tags:-

    The Unpaired tag or an empty tag which means that it has no end tag.

  • The <br> tag inserts a single line break.
  • The <hr> tag inserts a single thematic break(horizontal rule/line that is used to separate content).
