Basics Tags In HTML

Size of Headers:

px=pixels , 1em=16px , em=emphasis

  1. h1 = 32px 2em
  2. h2 = 24px 1.5em
  3. h3 = 20.8px 1.3em
  4. h4 = 16px 1em
  5. h5 = 12.8px 0.8em
  6. h6 = 11.2px 0.7em
  7. p(paragraph tag) = 16px 1em

Anchor Tag

Anchor tag defines a "Hyperlink" ,which is used to link from one page to another.
Syntax: Name to be appeared
If we use "=" within a tag it represents the tag's attribute
Syntax : Keyword = " Value"

Image Tag

Image tag is used to embed an image in a HTML page
Syntax :

  1. src (source) = File path with extention
  2. alt = Alternate text for an image
  3. width = Size in px or vh(visual height)
  4. hieght = Size in px or vh(visual height)
